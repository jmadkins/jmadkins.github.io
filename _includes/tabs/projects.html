Hubble

Closed Source

Hubble is an easy-to-use platform to help political campaigns centralize their voters, contacts, finances, and everything else.

Started in 2019 as the database my website used to store form submissions. Hubble grew into a powerhouse after a friends congressional campaign implemented it.

Hermes

Open Source

Rails API for logging emails and then sending them to AWS Simple Email Service for delivery. Supports multiple API clients and tracking emails sent by each client.

I created it for use at MythCoders but it's a great tool for any person/org that needs to log emails before sending them to AWS.
